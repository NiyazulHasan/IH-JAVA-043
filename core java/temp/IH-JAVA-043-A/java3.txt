> Standalone app
-----------------
A normal java program which contains main method is called standalone application.

ex:
	class Test
	{
		public static void main(String[] args)
		{
			-
			- //code to be execute 
			-		
		}
	}

> Desktop app
----------------
It is a software application or a program which is designed to perform perticular task.
ex:
	Recycle Bin 
	Control Panel 
	VLC Media player 
	and etc.

> Two-tier app 
----------------
Having more then one tier is called two-tier application.

Diagram: java3.1
Diagram: java3.2


> Distributed app
------------------
Diagram: java3.3

In client server architecture if multiple clients sending the request to the main server then main server will distribute the request it's parallel servers to reduce the burdun of main server such type of applications is called distributed application.


> Enterprise app
----------------
An application which deals with large business complex logic with the help of middleware services is called enterprise  application.

Here middleware services means authentication, autherization, malware production, security and etc.
ex:
	facebook
	online shopping website 
	and etc.
		
> ERP app 
----------
ERP stands for Enterprise Resource Planning.

ERP application is used to maintain the data in a enterpise.

Diagram: java3.4


> N-tier app
-----------
Having more then two tier is called n-tier application.


> Mobile app 
------------
It is a software application which is developed for wireless network devices like phone,cell, tab, cellular and etc rather then laptop's and pc's.

ex:
	PubG
	Gpay
	temple run 
	and etc.
